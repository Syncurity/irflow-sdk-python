irflow-sdk-python
=================

A python client for Syncurity IR-Flow REST API
----------------------------------------------

Installation
~~~~~~~~~~~~

``pip install irflow_client``

Upgrade
~~~~~~~

``pip install irflow_client --upgrade``

Build from source
~~~~~~~~~~~~~~~~~

| ``git clone git@github.com:Syncurity/irflow-sdk-python.git``
| ``pip install -e .``

Contribute
~~~~~~~~~~

Pull requests are always appreciated

Support
~~~~~~~
